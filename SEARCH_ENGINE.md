# Unified Search Engine Architecture

Aggregated search indexes and database routing.

---

## 1. Unified Router Scans
Our unified search router queries:
- Projects
- Conversations
- Messages
- Uploaded Files
- Semantic Knowledge Chunks (Qdrant Point vectors)

---

## 2. Match Scores
Results are ranked and sorted by textual relevance or vector similarity score, helping users locate resources instantly.
