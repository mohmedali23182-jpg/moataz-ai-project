# Moataz AI — User Guide

Get started using Moataz AI to configure isolated chats and RAG knowledge bases.

---

## 1. Onboarding & Dashboard Wizard
On your first login, complete the Onboarding wizard from the home screen:
1. Navigate to **Settings** to add your provider credentials.
2. Under **Projects**, create an isolated Workspace Project.
3. Open **Files** to upload and embed documentation manuals into the vector index.

---

## 2. Interactive Workspace Features
- **Chat Page**: Select your active project and model (e.g. OpenAI GPT-4o, Claude 3.5, Gemini) to query.
- **RAG citations**: When querying files, the system injects top matching document fragments and displays source citations.
- **Keyboard Shortcut**: Press `⌘K` or `Ctrl+K` to toggle the search command palette.
